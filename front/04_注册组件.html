# 注册组件

vue 注册组件的两种方式

# 局部注册

在对应页使用 components 注册组件。

<template>
  <count-to :startVal='startVal' :endVal='endVal' :duration='3000'></count-to>
</template>

<script>
import countTo from 'vue-count-to';
export default {
  components: { countTo },
  data () {
    return {
      startVal: 0,
      endVal: 2020
    }
  }
}
</script>

# 全局注册

在 @/main.js 文件下注册组件。

import countTo from 'vue-count-to'
Vue.component('countTo', countTo)

<template>
  <count-to :startVal='startVal' :endVal='endVal' :duration='3000'></count-to>
</template>
