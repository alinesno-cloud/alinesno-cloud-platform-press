# 通过重置密码

# 概要

重置密码主要用于批量操作，当需要批量账户初始化或者账户密码遗忘时，可通过单点管理平台一键重置密码功能对账户密码进行重置。

# 使用说明

进入单点登录管理平台，用户管理列表内对选中的账户点击重置按钮即可，重置密码值设定于单点配置列表， 可通过修改配置来修改重置密码值，重置后的密码会被指定加密方式进行加密。
